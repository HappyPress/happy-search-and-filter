== Description ==

Happy Search and Filter is a versatile plugin that provides advanced search and filter capabilities for custom post types. It includes Gutenberg blocks and shortcode support, allowing you to easily integrate search and filter functionality into your WordPress site. The plugin also offers various configuration options through the settings page.

== Installation ==

1. Upload the `happy-search-and-filter` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the settings under `Settings -> Happy Search and Filter`.
4. Use the `[hsf_search_filter]` shortcode or the Gutenberg block to add the search and filter form to your pages or posts.

== Frequently Asked Questions ==

= How do I add the search and filter form to my site? =
You can use the `[hsf_search_filter]` shortcode or the Gutenberg block to add the search and filter form to your pages or posts.

= Can I customize the search and filter fields? =
Yes, you can configure the search and filter fields through the settings page under `Settings -> Happy Search and Filter`.
